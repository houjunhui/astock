# A股超短学习计划（短线炒作）

## 目标
3个月内具备独立判断短线标的的能力，6个月形成稳定的超短交易体系。

---

## 第一阶段：基础（Day 1-30）

### 1.1 市场基本功
- [ ] 交易日历：熟悉A股交易时间（9:15集合竞价/9:30开盘/11:30午休/13:00开盘/15:00收盘）
- [ ] 集合竞价规则：9:15-9:20可以申报可以撤单，9:20-9:25只能申报不能撤单，9:25-9:30撮合收盘价
- [ ] 涨跌停制度：普通股票±10%，ST±5%，科创/创业板±20%
- [ ] 板位概念：1板（首板）/ 2板 / 3板 / 4板 / 5板+（高位板）/ 妖股（6板以上）
- [ ] 晋级/断板/续涨三种结局

### 1.2 涨停板基础
- [ ] 涨停板生态：为什么涨停、涨停的目的（吸筹/拉升/出货）
- [ ] 涨停原因分类：题材驱动 / 公告利好 / 资金运作 / 跟风
- [ ] 怎么看涨停原因：财联社/淘股吧/同花顺
- [ ] 首板 vs 连板：首板胜率~22%，连板晋级率随板位递减（但妖股除外）
- [ ] 一字板：高开封板/昨日涨停+今日高开 → 断板风险极高

### 1.3 技术指标入门
- [ ] RSI：超买>75/超卖<30，RSI>75历史断板率22%
- [ ] VR（成交量比率）：缩量/放量判断，vr<0.5反而安全（缩量不易断板）
- [ ] MACD：多头/空头/背离
- [ ] MA20/MA60：价格与均线的位置关系
- [ ] 趋势通道：上升/下降/横盘

### 1.4 每日复盘练习
- [ ] 每天收盘后记录：今日涨停数量/炸板率/跌停数量/市场温度
- [ ] 复盘昨日涨停股：第二天的开盘竞价/结局（晋级/断板/续涨）
- [ ] 建立自己的短线股票池笔记

---

## 第二阶段：短线核心（Day 31-60）

### 2.1 竞价情绪判断
- [ ] 竞价量能：竞价成交额 vs 昨日成交额，放量/缩量
- [ ] 竞价价格：高开/低开/平开，高开>5%为过热信号
- [ ] 竞价封单：涨停封单金额，封单不足<500万易炸板
- [ ] 昨日涨停股今日竞价：判断是否走一字板
- [ ] 情绪周期：回暖确认/退潮/冰点/恐慌

### 2.2 板位战法
- [ ] 首板（1板）：胜率~22%，适合入门练习，关注板块联动
- [ ] 2板：晋级率22%，有首板筛选基础后操作
- [ ] 3板：晋级率21%，3板是分歧点，竞价>4%易炸
- [ ] 4板+：高风险，4板×0.5系数，5板+考虑妖股模式
- [ ] 妖股基因：6板以上、断板率0%、博弈性质
- [ ] 一字板判断：昨日涨停+今日高开>5% → 断板风险>60%

### 2.3 板块联动
- [ ] 板块效应：同一板块≥3只涨停 → 联动效应强
- [ ] 强势板块：量价齐升的板块（电气设备/光伏/储能等）
- [ ] 弱势板块：量价齐跌（房地产/煤炭/银行）
- [ ] 题材周期：龙头股 → 板块扩散 → 跟风 → 退潮
- [ ] 板块轮动节奏

### 2.4 风险识别
- [ ] 断板信号：竞价低开/RSI>75超买/一字板高开
- [ ] 炸板率>40%：退潮信号
- [ ] 跌停数量>30只：市场极弱
- [ ] 市场温度<20：冰点，休息为主
- [ ] 高位板风险：4板+在退潮期加倍谨慎

---

## 第三阶段：实战（Day 61-90）

### 3.1 选股系统
- [ ] 竞价筛选条件：竞价区间/量能/板块联动/昨日涨停
- [ ] 只做最强板块：电气设备/新型电力/储能等
- [ ] 首板优先：1板在冰点市场晋级率仍高于连板
- [ ] 规避标的：一字板高开/竞价过热/历史封板率<70%
- [ ] 操作标的不超过3只

### 3.2 买入策略
- [ ] 竞价区间买入：1-3%区间最佳，过高不追
- [ ] 仓位管理：冰点市场轻仓30%，主升市场正常仓位
- [ ] 止损原则：断板即卖出，不抄底
- [ ] 持板时间：超短隔日，不超过3天

### 3.3 卖出策略
- [ ] 续涨卖出：第二天冲高即卖，不贪
- [ ] 断板止损：跌破涨停价即卖
- [ ] 到期强制卖出：持有不超过3个交易日
- [ ] 止盈参考：涨幅>15%可考虑分批卖出

### 3.4 交易记录
- [ ] 每笔交易记录：买入日期/价格/理由/预期/结局
- [ ] 每周总结：正确率/盈亏比/失败原因
- [ ] 每月复盘：哪些模式成功率高/哪些要放弃

---

## 第四阶段：体系固化（Day 91-180）

### 4.1 量化体系
- [ ] 建立自己的概率模型（可用模型辅助判断）
- [ ] 历史数据回测：验证自己的选股条件
- [ ] 概率校准：预测概率 vs 实际概率对比
- [ ] 期望值计算：晋级赚多少 vs 断板亏多少

### 4.2 心理管理
- [ ] 止损不手软：断板就是断板，不幻想
- [ ] 不频繁交易：超短机会不是每天有
- [ ] 冰点休息：市场差时空仓不亏钱
- [ ] 避免一致性：别人恐慌时保持冷静

### 4.3 持续进化
- [ ] 每日复盘雷打不动
- [ ] 关注政策/行业动态
- [ ] 跟自己的交易记录学习
- [ ] 每年更新一次模型系数
