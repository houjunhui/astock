# A股超短预测模型训练计划

## 现状评估

当前模型：
- XGBoost 3.2.0 + sklearn 1.8.0
- 样本量：26,228条（2024-01 至 2026-03）
- AUC-ROC：0.610
- 特征：rsi / log_vr / lb / trend_up / above_ma20 / jb_prob / rsi_high / rsi_low / lb_high / lb_mid

待解决问题：
- AUC偏低（0.61），区分度不足
- 特征缺少竞价数据/板块联动/市场情绪
- 无真正的概率校准（预测值≠真实概率）
- 一字板/妖股等特殊形态未单独建模

---

## 第一阶段：数据基础（Week 1-2）

### 1.1 特征工程增强
新增特征方向：
- [ ] **竞价特征**：`auction_chng`（竞价涨跌幅）/ `auction_bid`（封单金额）/ `auction_volume_ratio`（竞价量比）
- [ ] **板块特征**：板块内涨停数量 / 板块强弱信号（1/0/-1）
- [ ] **市场情绪特征**：市场温度 / 炸板率 / 涨跌家数比 / 昨日涨停今日炸板率
- [ ] **历史特征**：个股历史晋级率 / 个股历史封板率 / 个股涨停次数
- [ ] **筹码特征**：换手率 / 量能变化（今日vs昨日）
- [ ] **时间特征**：周一效应（周一涨停次日表现）/ 尾盘封板标识

### 1.2 数据清洗
- [ ] 剔除一字板无量涨停（正常市场不应有无量一字板）
- [ ] 剔除ST股/新股（上市<3个月）
- [ ] 剔除涨跌停当日成交额<500万的标的（流动性不足）
- [ ] 标注妖股样本（6板+，单独训练）

### 1.3 标签定义
- [ ] 晋级（正样本）：次日继续涨停
- [ ] 断板（负样本）：次日未涨停且收盘跌停或跌幅>8%
- [ ] 续涨（中性）：次日涨但未涨停 → 暂不纳入二分类

## 第二阶段：模型改进（Week 3-5）

### 2.1 模型结构调整
当前：单一XGBoost，binary:logistic
改进方向：
- [ ] **Stacking**：XGBoost + LightGBM + CatBoost 三模型融合
- [ ] **概率校准**：预测值用Platt Scaling / Isotonic Regression 校准
- [ ] **特殊样本**：妖股单独建模（6板+，样本少但规律独特）

### 2.2 特征重要性重评
```
目标：特征重要性 top 5 达到 >60% 解释度
当前：trend_up 23.7% / RSI 19.0% / jb_prob 11.8% / log_vr 10.1% / above_ma20 9.4%
改进后目标：加入竞价特征后，jb_prob应提升至20%+
```

### 2.3 超参优化
- [ ] 使用Optuna进行贝叶斯超参搜索
- [ ] 交叉验证：5-fold time series split（注意时间顺序）
- [ ] 目标：AUC > 0.68，精确率 > 90%

## 第三阶段：概率校准（Week 6-8）

### 3.1 当前问题
```
模型输出 P(晋级) = 30%，但实际晋级率只有22%
说明模型系统性高估概率
```

### 3.2 校准方案
- [ ] **分桶校准**（已有）：jb_prob分桶→历史真实晋级率
- [ ] **Platt Scaling**：在验证集上拟合 sigmoid，将输出转为真实概率
- [ ] **Beta校准**：用Beta分布拟合预测误差
- [ ] **联合校准**：晋级概率 + 断板概率联动约束（jz+dz=1，续涨=1-jz-dz）

### 3.3 概率正确性验证
```
校准后检验：
P(晋级=20-25%)区间 → 实际晋级率应接近22.5%
P(晋级=25-30%)区间 → 实际晋级率应接近27.5%
```

## 第四阶段：特殊形态建模（Week 9-12）

### 4.1 妖股模型
- [ ] 样本定义：6板以上 或 5板且换手率>15%
- [ ] 特征：连续涨停天数 / 换手率变化 / 板块情绪 / 龙虎榜机构席位
- [ ] 单独模型：LightGBM（小样本高深度）
- [ ] 目标：妖股晋级率40%，断板率接近0%

### 4.2 一字板模型
- [ ] 样本定义：昨日涨停+今日高开>5%
- [ ] 特征：封单金额 / 竞价量比 / 市场温度 / 板块联动数量
- [ ] 目标：识别一字板后断板概率>60%

### 4.3 首板模型
- [ ] 首板样本量最大，单独训练
- [ ] 重点特征：板块效应 / 竞价区间 / 量能健康度 / 历史封板率
- [ ] 目标：首板晋级率>25%

## 第五阶段：回测与验证（Week 13-16）

### 5.1 回测框架
- [ ] 按月/季度滚动回测
- [ ] 考虑交易成本（佣金0.03%+印花税0.1%）
- [ ] 分不同市场环境回测：主升/复苏/退潮/冰点
- [ ] 对比：模型策略 vs 随机选股 vs 买入所有涨停

### 5.2 核心指标
```
目标：
- 夏普比率 > 1.5
- 最大回撤 < 15%
- 胜率 > 55%（晋级赚钱）
- 盈亏比 > 1.5（晋级赚>断板亏）
```

### 5.3 分阶段验证
- [ ] 历史回测：2024年全年（样本内）
- [ ] 样本外验证：2025年至今（样本外）
- [ ] 近期验证：最近3个月单独验证

## 第六阶段：上线与迭代（持续）

### 6.1 模型更新机制
- [ ] 每日用新数据增量训练（不需要全量重训练）
- [ ] 每月全量重训练+系数更新
- [ ] 重大政策变化后重新标注

### 6.2 监控指标
- [ ] 预测分布 vs 实际分布偏差
- [ ] AUC衰减检测（>5%下降触发告警）
- [ ] 特殊样本（妖股/一字板）预测准确率

### 6.3 A/B测试
- [ ] 新模型上线前与旧模型对比
- [ ] 保守策略：新模型预测与旧模型一致时才操作
