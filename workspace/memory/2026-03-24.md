## A股竞价选股系统 v3 — 关键修复

### 昨日涨停池过滤 Bug（重要！）

**问题**：脚本用今日涨停池（含昨日涨停延续+今日新晋涨停的混合池）做竞价过滤，导致华塑控股(000509)、湖南发展(000722)、富祥药业(300497)等昨日未涨停的股票被误选入。

**根因**：
- 今日涨停池 = 昨日涨停延续板 + 今日新晋涨停（昨日根本没涨停）
- 竞价选股逻辑只适用于「昨日涨停→今日竞价延续」的情形
- 昨日未涨停的股票，今日竞价强不代表晋级延续，不适用此策略

**验证数据**：
- 昨日涨停池(20260323): 28只
- 今日涨停池(20260324): 24只
- 其中19只是今日新晋涨停（不在昨日池）
- 只有5只是昨日涨停延续：华电辽能(7板)、中利集团(3板)、拓日新能(2板)、辽宁能源(2板)、雪浪环境(2板)

**修复方案**：在 `scripts/astock_auction_full.py` 的 STEP 2b 插入昨日涨停池过滤逻辑：
- 获取昨日 `get_ladder` 数据，建立 `yday_codes` 集合
- 判断 `level >= 2` 或 `code in yday_codes` → 有效候选
- 其余 → 排除，打印排除原因

**修复后结果**：
- 有效候选：5只（全是高位板）
- 可操作：仅华电辽能(600396) B级30%
- 其余4只C级放弃

## 规则优化（14:10）

### 修复1：退潮期高位板强制降C
- 教训：大胜达3板+退潮期→误判买入→亏损
- 修复：退潮/恐慌期，lb≥3板强制降C级0%，不做
- 代码：`astock/auction.py` 评级逻辑后追加退潮期高位板拦截

### 修复2：一字板判断联合封板率
- 教训：中利集团一字板高开被warn，但封板率58%≠必然断板
- 旧规则：>9%一字板高开→warn
- 新规则：
  - 封板率<70% → warn（抛压大致命）
  - 封板率70-85% → susp（严格控仓）
  - 封板率≥85% → ok（强势封板，高开合理）

### 修复3：买入方式逻辑（待固化到脚本）
- 竞价涨幅0-3%：竞价买入（9:15-9:25）或9:30开盘买
- 竞价涨幅3-7%：等回调至涨幅一半位置买入
- 竞价涨幅>7%：等回调至75%位置，超过原涨幅+2%放弃
- 一字板涨停：无法买入
