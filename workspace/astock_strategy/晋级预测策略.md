# 晋级预测策略 v1.0

> 基于K线图形 × 成交量 × 技术指标 × 身位高度 的综合晋级预测
> 脚本：`scripts/astock_predict_v1.py`
> ⚠️ 注意：今日竞价涨幅为预设为0%基准；实盘需传入真实竞价数据

---

## 一、预测体系架构

```
输入：昨日涨停股列表（东方财富AKShare）
     今日竞价涨幅（已知数据）
输出：每只股票的晋级概率（0~100%）
```

**评分维度（6个维度，满分100）**

| 维度 | 权重 | 数据来源 | 明日可用 |
|------|------|---------|---------|
| 竞价涨幅 | 已知 | 今日集合竞价 | ✅ |
| K线图形 | 技术分析 | 腾讯日K线 | ✅ |
| 成交量结构 | 技术分析 | 腾讯日K线 | ✅ |
| 身位高度 | 已知 | 东方财富连板数 | ✅ |
| 历史晋级率 | 统计 | 15日积累数据 | ✅ |
| 资金活跃度 | 估算 | 成交量变化 | ✅ |

## 七、局限性

1. **竞价数据**：今日为非交易日，竞价设为0%基准；实盘需传入真实竞价
2. **大单数据缺失**：免费API无法获取Level2大单净流入/净流出
3. **历史样本有限**：高位妖股（4进5+）历史样本仅个位数
4. **RSI滞后**：RSI基于昨日收盘价，今日盘中会变化
5. **周K/月K图形**：当前用日K线模拟，未做真实周/月K聚合

## 七、高度预测模块（新增）

给定一只正在连板的股票，预测其最终能达到的高度。

### 基础晋级概率（从历史数据）

| 当前高度 | 续涨至N+1板 | N+2板 | N+3板 |
|---------|------------|-------|-------|
| 1板 | 13% | 1.9% | 0.3% |
| 2板 | 15% | 2.3% | — |
| 3板 | 20% | — | — |
| 4板 | 29% | — | — |

### 高度调整系数

| 因子 | 系数 | 说明 |
|------|------|------|
| 极度缩量（量比<0.5） | ×1.4 | 主力控盘锁筹 |
| 缩量（量比<0.8） | ×1.2 | 惜售明显 |
| 上升通道 | ×1.3 | 趋势向上 |
| 下降通道 | ×0.5 | 趋势向下 |
| 爆量（量比>3） | ×0.6 | 高位分歧 |
| MACD多头 | ×1.2 | 动能充足 |
| MACD空头 | ×0.7 | 动能不足 |
| 股价>MA20 | ×1.15 | 站均线 |
| 股价<MA20 | ×0.7 | 均线下方 |
| 妖股+爆量 | ×0.4 | 主力出货 |
| 妖股RSI>90 | ×0.3 | 极端见顶 |

### 输出说明

- **期望高度**：概率加权均值（加权所有可能高度的期望值）
- **最可能板数**：概率最高的那个高度
- **置信度**：高/中/低（取决于概率集中度）

脚本：`scripts/astock_height_predictor.py`

## 八、大单数据说明

真正有价值的"大单净流入/流出"需要Level2数据（年费数千元）。免费替代方案：

| 替代指标 | 计算方式 | 有效性 |
|---------|---------|--------|
| 成交量波动 | 量比 = 今日量/5日均量 | 中等 |
| 价格波动率 | 日内振幅 | 中等 |
| On-Balance Volume | 累计成交量×价格变化方向 | 弱 |

如需精准大单数据，可考虑：
- **掘金量化**（免费Level2）
- **Tushare Pro**（有资金流字段但需积分）
- **Wind/Choice**（付费）
