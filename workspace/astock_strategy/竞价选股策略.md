## 六、实战决策树 v3.0

```
已知某股昨日涨停，判断今日是否参与：

① 今日竞价 < 0%？ → 一票否决（全局通用）
② 今日竞价 0~2%？
   → 1进2：胜率8%，要求额比<0.5或量比<0.5才参与
   → 2进3：胜率8%，同标准
   → 3进4：胜率0%，直接放弃
③ 今日竞价 2~5%？
   → 1进2：胜率23%，量比<2.0 + 额比>1.0 可参与
   → 2进3：胜率23%，量比<2.0 + 额比>1.0 可参与
   → 3进4+：样本少，优先放弃
④ 今日竞价 5~9%？
   → 1进2：胜率32%，量比<2.0 可参与
   → 2进3：样本4中0胜，胜率0%，放弃
   → 3进4+：谨慎
⑤ 量比 > 1.6？ → 警告！降到10%胜率，减仓或放弃
⑥ 额比 0.8~1.0？ → 一票否决！6%胜率
⑦ 极度缩量（量比<0.5或额比<0.5）？ → 🏆 直接重仓

【介入门槛】
  综合满足：竞价>0% + 量比<1.6 + 额比不在0.8~1.0区间 + 非一字板
  → 正常参与
  加分项：量比<0.5 或 额比<0.5 → 可加重仓
```

## 八、策略局限性（诚实评估）

1. **样本时间集中**：数据仅覆盖2026年2月~3月，上升市特征明显
2. **高位身位样本少**：3进4仅15条，4进5仅7条，置信度不足
3. **一字板干扰**：一字板占比高，但其无法产生实际买点
4. **无板块联动**：东方财富数据缺少板块跟风数量字段
5. **成交额估算**：以"收盘价×成交量"替代，忽略盘中波动
